# 마케팅 사이트 개발 플랜

## 개요

하이록스(Hyrox) 스타일의 고강도 피트니스 마케팅 사이트를 개발합니다. 블랙 배경에 Volt Yellow 포인트 컬러를 사용한 강렬한 브랜드 아이덴티티를 구현하며, Next.js App Router의 Route Groups 기능을 활용하여 마케팅 사이트를 독립적으로 구성합니다.

## 아키텍처

```mermaid
graph TB
    subgraph AppRouter [app/ Directory]
        Root[layout.tsx<br/>Root Layout]
        
        subgraph Marketing [marketing Route Group]
            MarketingLayout[layout.tsx<br/>Marketing Layout]
            HomePage[page.tsx<br/>Landing Page]
            MarketingLayout --> HomePage
        end
        
        subgraph Auth [Existing Auth Routes]
            Login[/login]
            Signup[/signup]
            Account[/account]
        end
        
        Root --> Marketing
        Root --> Auth
    end
    
    subgraph Components [components/]
        MarketingHero[Hero Section]
        MarketingService[Service Intro]
        MarketingSocial[Social Proof]
        MarketingFooter[Footer]
    end
    
    HomePage --> MarketingHero
    HomePage --> MarketingService
    HomePage --> MarketingSocial
    HomePage --> MarketingFooter
```



## 1. 프로젝트 구조 설정

먼저 Route Groups를 사용하여 마케팅 사이트를 독립적으로 분리합니다.

### 디렉토리 구조

```javascript
app/
├── (marketing)/           # 마케팅 사이트 그룹
│   ├── layout.tsx         # 마케팅 전용 레이아웃
│   └── page.tsx           # 랜딩 페이지 (/)
├── login/                 # 기존 인증 페이지 유지
├── signup/
├── account/
└── layout.tsx             # 루트 레이아웃
```

**작업 내용:**

- `app/(marketing)/layout.tsx` 생성: 마케팅 사이트 전용 레이아웃
- `app/(marketing)/page.tsx` 생성: 메인 랜딩 페이지 (Hero + Service Intro + Social Proof + Footer)

## 2. 브랜드 테마 설정

하이록스 브랜드 컬러와 타이포그래피를 Tailwind CSS에 정의합니다.

### Tailwind 설정 업데이트

[`app/globals.css`](app/globals.css)에 하이록스 브랜드 컬러를 CSS 변수로 추가:

```css
@layer base {
  :root {
    --volt-yellow: 210 255 0;     /* #D2FF00 */
    --pure-black: 0 0 0;           /* #000000 */
    --dark-grey: 26 26 26;         /* #1A1A1A */
    --light-grey: 161 161 170;     /* #A1A1AA */
  }
}
```



### 폰트 설정

강렬한 타이포그래피를 위해 Extra Bold 웨이트를 추가합니다.

## 3. 공통 컴포넌트 개발

재사용 가능한 마케팅 컴포넌트를 작성합니다.

### 3.1 Section Container

모든 섹션에서 사용할 공통 컨테이너 컴포넌트를 생성합니다.**파일:** `components/marketing/section-container.tsx`**기능:**

- 일관된 패딩 및 마진
- 반응형 최대 너비
- 선택적 배경색

### 3.2 Animated Element

스크롤 시 페이드인 애니메이션을 적용하는 컴포넌트를 생성합니다.**파일:** `components/marketing/animated-element.tsx`**기술:**

- Framer Motion 사용
- Intersection Observer로 뷰포트 진입 감지
- 커스터마이징 가능한 애니메이션 옵션

### 3.3 CTA Button

Volt Yellow 포인트 컬러를 사용한 CTA 버튼을 생성합니다.**파일:** `components/marketing/cta-button.tsx`**스타일:**

- 배경: Volt Yellow (#D2FF00)
- 텍스트: Black
- Hover 효과: 살짝 어두워지는 효과
- 모바일/데스크톱 반응형 크기

## 4. Hero Section 개발

첫인상을 결정하는 가장 중요한 섹션입니다.

### 구성 요소

1. **풀스크린 배경**

- 검은 배경에 그라디언트 오버레이
- 플레이스홀더: Unsplash의 고강도 운동 이미지

2. **메인 카피**
   ```javascript
      당신의 한계는 매일 갱신된다.
      Your limits are renewed every day.
   ```




3. **서브 카피**
   ```javascript
      단순한 운동을 넘어 체력의 한계에 도전하세요.
      하이록스 스타일의 고강도 트레이닝 프로그램과
      전문 코칭이 당신을 기다립니다.
   ```




4. **CTA 버튼**

- Primary: "지금 도전하기" → `/signup`으로 연결
- Secondary: "프로그램 보기" → Service Intro 섹션으로 스크롤

### 기술 구현

**파일:** `components/marketing/hero-section.tsx`**기능:**

- Next.js `Image` 컴포넌트로 최적화
- 텍스트 애니메이션 (타이핑 효과 또는 페이드인)
- 스크롤 다운 인디케이터
- 모바일에서는 텍스트 크기 자동 조정

## 5. Service Intro Section 개발

서비스의 차별점을 3-4개 포인트로 보여주는 섹션입니다.

### 구성 요소

1. **섹션 헤드라인**
   ```javascript
      측정 가능한 결과, 확실한 변화.
   ```




2. **3가지 핵심 포인트**

**Systematic Program**

- 아이콘: 체크리스트 또는 그래프
- 설명: "하이록스 기반의 검증된 8단계 훈련 커리큘럼"

**Daily Coaching**

- 아이콘: 채팅 또는 코치
- 설명: "매일 주어지는 과제와 코치의 실시간 피드백"

**Data Driven**

- 아이콘: 차트 또는 카메라
- 설명: "사진과 수치로 기록되는 당신의 피트니스 히스토리"

### 레이아웃

- 데스크톱: 3단 그리드
- 태블릿: 2단 그리드
- 모바일: 1단 세로 스택

### 기술 구현

**파일:** `components/marketing/service-intro-section.tsx`**컴포넌트:**

- `FeatureCard`: 각 포인트를 표시하는 카드 컴포넌트
- lucide-react 아이콘 사용
- Hover 시 Volt Yellow 보더 하이라이트

## 6. Social Proof Section 개발

신뢰도를 구축하는 고객 후기 섹션입니다.

### 구성 요소

1. **섹션 헤드라인**
   ```javascript
      이미 수천 명이 자신의 기록을 깨고 있습니다.
   ```




2. **고객 후기 슬라이더**

- 플레이스홀더: 3-5개의 샘플 후기
- 각 후기: 이름, 프로필 사진, 별점, 텍스트
- 자동 슬라이드 (5초 간격)

3. **통계 카운터** (선택 사항)
   ```javascript
      총 훈련 완료 횟수: 50,000+
      활동 중인 회원: 1,200+
      평균 만족도: 4.8/5.0
   ```




### 기술 구현

**파일:** `components/marketing/social-proof-section.tsx`**라이브러리:**

- Embla Carousel React (이미 설치됨)
- 자동 플레이 + 인디케이터 도트
- 터치 스와이프 지원 (모바일)

## 7. Footer 개발

법적 정보와 링크를 포함한 푸터입니다.

### 구성 요소

1. **왼쪽: 브랜드 정보**

- 서비스 이름 로고
- 짧은 설명

2. **중앙: 링크**

- 서비스 소개
- 프로그램
- 고객 후기
- 로그인

3. **오른쪽: 법적 정보**

- 사업자 정보 (플레이스홀더)
- 이용약관
- 개인정보처리방침

4. **하단: 소셜 미디어 & 저작권**

- Instagram, YouTube, Facebook 링크 (플레이스홀더)
- © 2025 [서비스명]. All rights reserved.

### 기술 구현

**파일:** `components/marketing/footer.tsx`**스타일:**

- 배경: Dark Grey (#1A1A1A)
- 텍스트: Light Grey
- 링크 Hover: Volt Yellow

## 8. SEO 최적화

검색 엔진 최적화를 위한 메타데이터 설정입니다.

### Metadata 설정

**파일:** `app/(marketing)/layout.tsx`

```typescript
export const metadata: Metadata = {
  title: "고강도 피트니스 트레이닝 | [서비스명]",
  description: "하이록스 스타일의 검증된 8단계 훈련 커리큘럼. 매일 주어지는 과제와 전문 코치의 실시간 피드백으로 당신의 한계를 갱신하세요.",
  keywords: ["피트니스", "하이록스", "트레이닝", "코칭", "운동 프로그램"],
  openGraph: {
    title: "당신의 한계는 매일 갱신된다",
    description: "고강도 피트니스 트레이닝 프로그램",
    images: ["/og-image.jpg"], // 추후 추가
  },
};
```



### 구조화된 데이터

JSON-LD 스키마를 추가하여 검색 결과에 리치 스니펫이 표시되도록 합니다.

## 9. 성능 최적화

### 이미지 최적화

- Next.js `Image` 컴포넌트 사용
- WebP 포맷 사용
- 적절한 sizes 속성 설정
- 우선순위 이미지(Hero)에 `priority` 속성

### 코드 분할

- Framer Motion은 동적 import로 로드
- 각 섹션을 별도 컴포넌트로 분리

### 폰트 최적화

- `next/font`로 폰트 최적화
- 폰트 서브셋팅 (latin만 로드)

## 10. 반응형 디자인

### 브레이크포인트

- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

### 모바일 우선 전략

모든 컴포넌트를 모바일 뷰부터 설계하고 데스크톱으로 확장합니다.

## 플레이스홀더 콘텐츠

실제 콘텐츠가 준비될 때까지 다음을 사용합니다:

- **이미지**: Unsplash API 또는 로컬 샘플 이미지

- **카피라이팅**: PRD에 제시된 텍스트 사용
